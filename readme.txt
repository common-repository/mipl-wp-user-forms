== Description ==

Simple Login And Registration Forms, Users=>Login & Registration".

= Features =

* Create Registration Forms.
* Custom Fields to Registration Forms.
* Create Login Form.
* Social Media Google Login.


== Installation ==

1. Upload plugins folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Create Rregistration Form. 
4. Update the Users Forms settings.
5. Setup the Google Login keys.
6. Setup the Recaptcha keys.
7. Add Login & Registration Form shortcodes to page.
